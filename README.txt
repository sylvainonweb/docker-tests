* docker images  : liste les images docker
* docker build . -t <NOM IMAGE> : compile le fichier Dockerfile se trouvant dans le répertoire actuel et lui affecte un nom
* docker image rm <ID IMAGE> : supprime l'image dont l'id est spécifié
* docker run <NOM IMAGE> : démarrer un conteneur basé sur l'image spécifiée
* docker ps : liste les conteneurs docker
